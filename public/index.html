Get Started

1. docker run -p 4242:4242 zer0liquid/the-agents-hub:latest
2. Add to Claude Code: npx the-agents-mcp@latest --hub http://localhost:4242
3. Add to CLAUDE.md: Call mcp__agent-visualizer__update_state before every work phase.
4. Open localhost:4242/viewer
